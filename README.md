## Overview
This repository contains my hands-on work and notes from the **5-Day AI Agents Intensive Course with Google**, delivered through Kaggle Learn. The course is a practical, end-to-end program focused on designing, building, evaluating, and deploying AI agents using modern agentic architectures.

## What This Course Covers
The course explores the full lifecycle of AI agents, including:

- Foundations of AI agents and agentic architectures  
- Tool usage and interoperability using **Model Context Protocol (MCP)**  
- Context engineering with sessions and long-term memory  
- Agent quality, observability, logging, tracing, and evaluation  
- Moving from prototypes to production-ready, scalable **multi-agent systems** using the **Agent2Agent (A2A) Protocol**
